# Coffee Shop Sales Dashboard

## Project Overview
This project analyzes coffee shop sales data using Excel dashboards and data analysis techniques.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Data Cleaning

## Key Insights
- Identified top-performing products
- Analyzed peak sales hours
- Compared category-wise sales performance
- Tracked revenue trends

## Features
- Interactive dashboard
- Slicers and filters
- KPI cards
- Trend analysis
